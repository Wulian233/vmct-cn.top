| 模组加载器        | 支持的Minecraft版本                    |
| ---------------- | ------------------------------------- |
| Forge            | 1.12/1.16.5/1.18~1.20.4①              |
| NeoForge         | 1.20~1.20.6                           |
| Fabric/Quilt②    | 1.16.5~1.20.6                         |

注①：对于Forge在1.20.5及以上，由于模组依赖Cloth config API已不支持，故本模组也不支持。

注②：我们不提供专门的代码支持，但可以在Quilt上正常工作。
